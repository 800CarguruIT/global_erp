# API Documentation

Keep endpoint contracts and conventions in this section.

## Contents

- `endpoint-template.md`
- `conventions.md` (recommended)
- `<service-or-module>-api.md`

## Minimum Requirements per Endpoint

- URL and method
- Authentication/authorization requirement
- Request schema and example
- Response schema and example
- Error responses
- Idempotency and rate-limiting notes (if applicable)
