# Deployment

Store production and staging runbooks here.

## Include

- Deployment steps
- Migration strategy
- Rollback procedure
- Health checks
- Post-deploy verification checklist
- Incident escalation contacts

## Deployment Checklist

1. Confirm changelog and migration notes.
2. Deploy application and run migrations.
3. Execute smoke tests.
4. Monitor logs and metrics.
5. Confirm business-critical flows.
